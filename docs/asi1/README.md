# ASI1 - Modelo de IA Local

> ⚠️ **IMPORTANTE:** ASI1 é um modelo LOCAL, NÃO é OpenAI!

---

## O que é ASI1?

ASI1 é um modelo de IA que roda **localmente** via **Ollama**.

**NÃO é:**
- ❌ OpenAI
- ❌ API externa
- ❌ Serviço cloud
- ❌ GPT-4 ou qualquer modelo da OpenAI

**É:**
- ✅ Modelo local (roda na sua máquina)
- ✅ Via Ollama
- ✅ Privado e offline
- ✅ Sem custos de API

---

## Por que ASI1?

- **Privacidade:** Dados não saem da máquina
- **Custo:** Sem gastos com API
- **Controle:** Modelo customizável
- **Latência:** Resposta rápida (local)

## Uso no Neobot

ASI1 é usado para:
- Análise de código local
- Geração de documentação
- Tarefas que não requerem modelos grandes
