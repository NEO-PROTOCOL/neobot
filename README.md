▓▓▓ WHAT IS NEO PROTOCOL?
────────────────────────────────────────
NEØ.BOT is a decentralized AI assistant
control plane that runs on your infra,
with your rules, following Web3 laws.

Born from Moltbot's industrial foundation.
Hybrid architecture:

- 30% Neobot Core
- 70% NEO Layer

```text
[####] NEO Layer (60%) ............. OK
[####] Neobot Core (40%) ........... OK
```
